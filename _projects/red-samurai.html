Red SamuraiI

A Samurai-themed Custom Keycap Set

DISCUSS ON GEEKHACK

Base Kit

The base kit contains enough keys to support several keyboard layouts.

Hiragana Alphas Kit - Gaijin

The Gaijin alphas kit provides a taste of Japanese culture with both Latin and Hiragana legends.

Hiragana Alphas Kit - Hiragana

The Hiragana alphas kit provides a taste of Japanese culture with Hiragana monolegends.

Novelties Kit

The Novelties kit contains samurai and anime/ninja-inspired novelties, along with VIM keys for programmers and alternate color arrow keys!

Norde Kit

The Norde kit provides a good amount of Danish, Finnish, German, Norwegian, and Swedish characters for foreign layouts.

DC Kit

The DC kit contains the keys needed for the Dvorak and Colemak layouts with homing barred and scooped H, U, N, T keys.

ErgoPlanck Kit

The ErgoDox and Planck blanks kit containing the modifiers for ErgoDox and Planck.

Spacebars Kit

The Spacebars kit includes an alternative color 6U Centered Stem(x1), 6U Off-Centered Stem(x1), 6.25U, 7U, and the basic spacebars in 1.75U, 2.00U, 2.25U, and 2.75U for split and 40% keyboards.
